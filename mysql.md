## 存储引擎
innoDB和myisam对比
| **特性** | **InnoDB **                                                                    | **MyISAM **                |
| -------- | ------------------------------------------------------------------------------ | -------------------------- |
| 事务     | 支持                                                                           | 不支持                     |
| 锁       | 行锁&&表锁                                                                     | 表锁                       |
| 缓存     | 不仅缓存索引还要缓存真实数据，对内存要求较高，而且内存大小对性能有决定性的影响 | 只缓存索引，不缓存真实数据 |
| 外键     | 支持                                                                           | 不支持                     |
| 索引类型 | 主键：聚簇索引                                                                 | 非聚簇索引                 |
| 备份     | 热备份                                                                         | 不支持                     |
